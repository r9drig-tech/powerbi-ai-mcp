Cena 1 — Receita Total
powerbi-ai-mcp · mcp_server.py
rodrigo ❯ Crie uma medida de Receita Total usando a coluna Valor_Venda
IA ❯ Medida gerada · pasta: 💰 Financeiro
Receita Total =
SUM(Vendas[Valor_Venda])
Soma todos os valores de venda da tabela Vendas. Pronta para uso no Power BI.
Cena 2 — Margem de Lucro
powerbi-ai-mcp · mcp_server.py
rodrigo ❯ Crie uma medida de Margem de Lucro em percentual
IA ❯ Medida gerada · pasta: 💰 Financeiro
Margem % =
DIVIDE(
[Lucro],
[Receita Total],
0 // fallback divisão por zero
)
Usa DIVIDE() com fallback 0 — nunca retorna erro em contexto vazio.
